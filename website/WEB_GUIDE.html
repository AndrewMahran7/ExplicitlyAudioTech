üöÄ Getting Started

Step 1: Start the Server

Open PowerShell/Command Prompt and run:

python -m explicitly web

Or use the run script:

python run_web.py

Step 2: Open Your Browser

Navigate to:

http://localhost:5000

Step 3: Upload Your File

Click to browse or drag & drop your audio file (MP3, WAV, FLAC, etc.)

Step 4: Choose Your Settings

  • Model: Select between Speed, Balanced, or Accuracy
  • Method: Choose Mute, Bleep, or Reverse
  • Device: Auto-detect or manually select GPU/CPU

Step 5: Process & Download

Click "Start Processing" and wait for completion. Then download your clean audio!

üìù Tips for Best Results

  • Use GPU acceleration when possible for 5-10x faster processing
  • Start with "Balanced" model for most use cases
  • Use "Speed" model for quick tests or long files
  • Use "Accuracy" model for important content
  • Keep files under 100MB for smoother uploads
  • Ensure good internet connection if downloading large files

‚ùì Troubleshooting

Server won't start

Port 5000 might be in use. Try a different port:

python -m explicitly web --port 8080

Processing is very slow

  • Check if GPU is being used (shown at startup)
  • Try the "Speed" model preset
  • Process shorter audio clips

Upload fails

  • Check file size (max 100MB)
  • Verify file format is supported
  • Try converting to MP3 or WAV first
